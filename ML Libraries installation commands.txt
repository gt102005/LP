Jupyter Notebook installation



1. System Update and Python Installation:
It is recommended to update your system and ensure Python 3 is installed, as Jupyter Notebook relies on it.
Code

sudo apt update && sudo apt upgrade -y
sudo apt install python3 python3-pip python3-venv -y
2. Creating and Activating a Virtual Environment (Recommended):
Using a virtual environment helps manage project dependencies and avoids conflicts with system-wide Python packages.
Code

mkdir my_jupyter_project
cd my_jupyter_project
python3 -m venv jupyter_env
source jupyter_env/bin/activate
3. Installing Jupyter Notebook:
Once your virtual environment is active, install Jupyter Notebook using pip.
Code

pip install notebook
4. Launching Jupyter Notebook:
Navigate to the directory where you want to store your notebooks and launch Jupyter Notebook.
Code

cd /path/to/your/notebooks_directory # Replace with your actual directory
jupyter notebook
This command will open Jupyter Notebook in your default web browser, typically at http://localhost:8888.
5. Stopping Jupyter Notebook:
To stop the Jupyter Notebook server, press Ctrl+C in the terminal where it is running. You may need to press it twice to confirm.







Run this instead:

python -m jupyterlab
